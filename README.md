# ak-portfolio

Personal portfolio website built with React, hosted on GitHub Pages at https://arnav149.github.io/ak-portfolio.

## Development

### `npm start`

Runs the app in development mode. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `npm run build`

Builds the app for production to the `build/` folder.

### `npm run deploy`

Builds the app and deploys it to the `gh-pages` branch via `git subtree`, which GitHub Pages serves automatically.
